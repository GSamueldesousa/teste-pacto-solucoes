# Teste Automatizado - Pacto Soluções

Projeto de teste end-to-end com **Playwright** para validar a busca por "Pacto Soluções" no Yahoo.

## Tecnologias
- Playwright
- JavaScript
- Node.js
- GitFlow

## Como executar
1. Clone o repositório
2. `npm install`
3. `npx playwright test`
4. `npx playwright show-report`
